# 경쟁하며 학습하는 GAN

* GAN 기초
    * 생성자와 판별자

    * 생성자와 판별자 구현
    * GAN 학습 구현
    * 결과물 시각화
* [cGAN으로 생성 제어하기](conditional_gan.ipynb)
